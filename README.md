# tgtools.core
通用工具封装
常用功能：
数据源管理（db）
后台服务管理（service 简单调度任务实现）
插件管理(plugin)
简单消息管理(message)
常用工具（util）
缓存（ehcache)
任务模型（task）
网络交互对象（net）
